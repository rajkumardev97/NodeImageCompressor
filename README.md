# NodeImageCompressor
Compress your images by imagemin

# Start Project

1)npm install

2)npm start

# PNG	            imagemin-pngquant	  imagemin-optipng
# GIF	            imagemin-giflossy	  imagemin-gifsicle
# SVG	            Imagemin-svgo	
# WebP	            imagemin-webp
